## How to run

Clone the repo

Run a script:

Interpreted languages (e.g. Python, Ruby):

python python.py
ruby ruby.rb

Compiled languages (e.g. C, Java):

gcc c.c -o c && ./c
javac java.java && java java
go run go.go

File naming convention:

Each file is named <language>.<extension> and contains a minimal “Hello, World!”.
